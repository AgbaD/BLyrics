## Requirements
1. Python3 
2. Terminal
3. [Requirements](requirements.txt)
4. An open mind

## Usage
1. Clone the repository
```
$ git clone 'https://github.com/BlankGodd/BLyrics.git'

$ cd BLyrics
```

2. Install the dependencies
```
$ pip3 install -r requirements.txt
```

3. Run script 
```
$ python3 main.py
```
