## Usage
1. Clone the repository
```sh
$ git clone https://github.com/BlankGodd/BLyrics.git

$ cd BLyrics
```
2. Setup virtual enviroment

3. Install the dependencies
```sh
$ pip3 install -r requirements.txt
```
4. Move into scripts directory
```sh
$ cd blyrics
```
5. Run script 
```sh
$ python3 main.py
```
